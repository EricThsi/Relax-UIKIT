# Typography 排版

## Font Family 字体家族

```css
font-family:
  font-family: Roboto, Helvetica, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', '微软雅黑', Arial, sans-serif;
;
```

## Base Font Size 基础字号

```css
font-size: 14px;
```

## Line Height 行高

```css
line-height: 1.5;
```

## Line Weight 字重

```css
font-weight: 400;
```

## Font Color 字体颜色

```css
color: #000;
```
